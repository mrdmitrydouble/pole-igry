# Поле игры

Интерактивная карта для пар в периоды **рассинхрона желания** — когда у одного партнёра либидо на подъёме, а у другого на спаде.

**→ Открыть карту: https://mrdmitrydouble.github.io/pole-igry/**

Два желания редко движутся в одной фазе: у каждого — свои рост и спад.

Карта собирает всё поле возможностей для такого периода: от практик наедине с собой до вариантов, требующих серьёзных договорённостей. Указаний она не раздаёт — это поле, которое изучают вдвоём. 85 вариантов, сгруппированных по зонам близости, с фильтрами по роли (рост / спад / вместе), анатомии пары и каналу восприятия.

## Как устроено

Одностраничный статический сайт. React встроен локально — внешних зависимостей в рантайме нет, страница работает автономно.

- `index.html` — страница
- `bundle.js` — приложение (собрано из `source.jsx`)
- `react.js`, `react-dom.js` — React 18 UMD (локально)
- `source.jsx` — исходник компонента

Пересборка после правок `source.jsx`:

```bash
npx esbuild source.jsx --jsx=transform --minify --outfile=bundle.js
```

(в `source.jsx` для сборки заменены `import ... from "react"` на глобальный `React` и добавлено монтирование в `#root`.)
